💇 Telegram-бот для салона красоты «Beauty»

Telegram-бот для онлайн-записи клиентов в салон красоты.

Клиент выбирает услугу, дату и время, оставляет имя и телефон. После этого запись сохраняется в базу данных, а владелец салона получает уведомление в Telegram.

👨‍💼 Возможности для владельца

Владелец салона может посмотреть все записи командой:

/bookings

Команда доступна только администратору, чей Telegram ID указан в переменной OWNER_ID.

⚙️ Запуск проекта локально
Клонировать репозиторий:
git clone ссылка_на_репозиторий
Перейти в папку проекта:
cd telegram-salon-bot
Создать виртуальное окружение:
python -m venv venv
Активировать виртуальное окружение:

Для Windows PowerShell:

venv\Scripts\Activate.ps1
Установить зависимости:
pip install -r requirements.txt
Создать файл .env и добавить переменные:
BOT_TOKEN=токен_бота
OWNER_ID=telegram_id_владельца
DB_DIR=.
Запустить бота:
python bot.py
🌐 Деплой

Бот задеплоен на Amvera и работает 24/7.

Для хранения базы данных используется постоянное хранилище /data, чтобы записи клиентов не удалялись после пересборки проекта.

Переменные окружения на сервере:

BOT_TOKEN=токен_бота
OWNER_ID=telegram_id_владельца
DB_DIR=/data
