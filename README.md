# 💇 Telegram-бот для салона красоты «Beauty»

Бот для онлайн-записи: клиент выбирает услугу, дату и время, оставляет контакты — запись сохраняется, а владелец получает уведомление.

## 🛠 Технологии
- Python
- aiogram 3.x
- SQLite (база данных записей)
- python-dotenv

## 🚀 Возможности
- Выбор услуги (стрижка, борода, маникюр, педикюр)
- Выбор даты и времени
- Сбор имени и телефона клиента
- Сохранение записей в базу данных
- Мгновенное уведомление владельцу о новой записи
- Команда /bookings — просмотр всех записей (для администратора)

## ⚙️ Запуск
1. Клонировать репозиторий
2. Создать виртуальное окружение и установить зависимости: `pip install -r requirements.txt`
3. Создать файл `.env` с переменными `BOT_TOKEN` и `OWNER_ID`
4. Запустить `python bot.py`
